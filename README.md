# tool-box

## 项目简介
一个小工具集项目

### 安装依赖
```
yarn install
```

### 运行项目
```
yarn serve
```

### 打包项目
```
yarn build
```

### 发布项目
```
yarn commit
```

### 在线地址
[https://zeroonbush.github.io/tool-box/#/](https://zeroonbush.github.io/tool-box/#/)
